This is my own stripped down version of angular-seed, for a really basic [AngularJS](http://angularjs.org/) web app.
Feel free to use it to quickly bootstrap your angular webapp projects WITHOUT the node server and test-environment and with all dependencies installed.


```bash
git clone --depth=1 https://github.com/Dynastig/angular-seed.git <your-project-name>
```

## Loading Angular Asynchronously

```
npm run update-index-async
```

This will copy the contents of the `angular-loader.js` library file into the `index-async.html` page.
You can run this every time you update the version of Angular that you are using.
